开发文档

了解项目架构、技术栈与扩展方法

📦 项目结构

周易算命/
├── frontend/
│   └── index.html      # 前端页面
├── src/
│   ├── core/           # 核心算法
│   │   ├── FortuneTeller.ts
│   │   └── ZhouyiMethod.ts
│   ├── bazi/           # 八字模块
│   │   ├── baZiCalculator.ts
│   │   ├── stemsAndBranches.ts
│   │   └── lunarConverter.ts
│   ├── liuyao/         # 六爻模块
│   │   └── liuYaoCalculator.ts
│   ├── fate/           # 命理分析
│   │   └── fateAnalyzer.ts
│   └── extensions/     # AI扩展
│       └── deepseek.ts
├── core.test.js        # 测试文件
├── package.json
└── tsconfig.json

🛠️ 技术栈

  • • TypeScript - 核心逻辑实现
  • • HTML/CSS/JS - 前端界面
  • • Node.js - 运行环境
  • • DeepSeek API - AI 智能解读
  • • Jest - 单元测试

🚀 快速开始

# 安装依赖
npm install

# 编译 TypeScript
npm run build

# 运行测试
npm test

# 启动前端
npx http-server frontend -p 8080 -c-1

📖 核心模块说明

  • • FortuneTeller - 周易64卦占卜核心
  • • BaZiCalculator - 四柱八字排盘
  • • LiuYaoCalculator - 六爻起卦解卦
  • • FateAnalyzer - 五行命理分析
  • • DeepSeek - AI 扩展接口
