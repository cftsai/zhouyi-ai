# 贡献指南

感谢你对周易算命 AI 项目的兴趣！欢迎参与贡献。

## 🤝 如何贡献

### 报告问题

如果你发现了 bug 或有功能建议，请提交 [Issue](https://github.com/your-username/zhouyi-ai/issues)。

提交 Issue 时请包含：
- 问题描述
- 复现步骤
- 预期行为
- 实际行为
- 运行环境（操作系统、Node.js 版本）

### 提交代码

1. **Fork** 本仓库
2. **创建分支** (`git checkout -b feature/amazing-feature`)
3. **提交更改** (`git commit -m 'Add amazing feature'`)
4. **推送分支** (`git push origin feature/amazing-feature`)
5. **创建 Pull Request**

### 代码规范

- 使用 2 空格缩进
- 使用 TypeScript 编写核心逻辑
- 保持代码风格与现有代码一致
- 添加必要的注释

### 提交信息格式

```
<type>: <description>

# type 包括：
# feat:     新功能
# fix:      修复 bug
# docs:     文档更新
# style:    代码格式调整
# refactor: 重构代码
# test:     测试相关
# chore:    构建/工具相关
```

### 开发流程

```bash
# 1. Fork 并克隆仓库
git clone https://github.com/your-username/zhouyi-ai.git
cd zhouyi-ai

# 2. 安装依赖
npm install

# 3. 创建分支
git checkout -b feature/your-feature

# 4. 开发
npm run dev

# 5. 运行测试
npm test

# 6. 编译检查
npm run build

# 7. 提交并推送
git add .
git commit -m "feat: add your feature"
git push origin feature/your-feature
```
