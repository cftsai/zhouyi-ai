# 周易AI命理框架 - 更新总结

## 更新日期：2026-04-17

## 一、核心算法修复

### 1. 天干地支排盘系统重构 (`src/bazi/stemsAndBranches.ts`)

**问题**：原代码月柱未按节气划分，年柱未以立春为界，不符合《渊海子平》规范。

**修复内容**：
- 新增 **24节气数据表**（2020-2026年），精确到日
- 新增 `getSolarTermMonth()` 函数，根据出生日期判断所处节气月份
- 新增 `isBeforeLiChun()` 函数，正确判断年柱以立春为界
- 修正 `getYearStemBranch()` 以立春为年柱分界
- 修正 `getMonthStemBranch()` 以节气为月柱分界
- 新增 **纳音表**（60甲子纳音完整数据）
- 新增 **天干阴阳属性表**
- 新增 **地支五行属性表**

### 2. 命理分析系统重构 (`src/fate/fateAnalyzer.ts`)

**问题**：原代码大运流年推算严重简化（固定值），五行喜忌分析过于简单，缺少六亲关系。

**修复内容**：

#### 大运推算
- 实现 **阳男阴女顺排，阴男阳女逆排** 的传统规则
- 实现 **从月柱起运** 的正确算法
- 实现 **起运年龄计算**（三日为一年法）
- 实现 **大运吉凶评估**（基于日主强弱与五行生克）

#### 五行旺衰分析
- 新增 **月令五行力量表**（12地支×5行）
- 实现 `calculateWuxingStrength()` 函数，综合计算五行力量
- 实现 **得令加权**（月令五行加成1.5倍）
- 实现 **身强身弱判断** 及对应的喜忌分析

#### 十神与六亲
- 完善 **十神映射表**（10天干×10天干，完整100组）
- 新增 **六亲关系映射**（比肩→兄弟、食神→子孙、正财→妻财、正官→官鬼、正印→父母）
- 新增 **地支藏干** 分析（12地支藏干完整数据）
- 实现 **十神力量统计**（含天干十神+地支藏干十神）

#### 流年推算
- 实现 **流年干支正确计算**（基于1984甲子年推算）
- 实现 **流年吉凶评估**（基于日主强弱与流年五行关系）

### 3. 周易起卦方法修复 (`src/core/ZhouyiMethod.ts`)

**问题**：原代码随机生成卦象，不符合传统起卦方法。

**修复内容**：
- 实现 **铜钱起卦法**（三硬币法，6/7/8/9四象）
- 实现 **动爻识别**（老阳9、老阴6为动爻）
- 新增 **64卦完整数据**（二进制编码→卦名映射）
- 新增 **64卦卦辞** 完整数据
- 新增 **乾/坤/泰/否四卦爻辞** 详细数据
- 修正卦象编码，消除重复键问题

### 2. RAG检索服务 (`src/rag/ragService.ts`)

**功能**：
- `searchRag(query, options?)`：基于关键词的RAG检索，支持按来源、分类过滤
- `augmentBaZiWithRag(baZiInfo, fateAnalysis)`：自动从八字信息提取关键词，检索相关经典文献
- `getRagStats()`：获取知识库统计信息
- 检索结果包含：匹配条目、增强分析文本、相关经典引用

### 3. 模块导出 (`src/rag/index.ts`)

统一导出RAG模块所有功能，通过 `src/index.ts` 对外暴露。
